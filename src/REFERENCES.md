## API Coverage
- **Accounts**: list ad accounts, get ad account details
- **Campaigns**: list, get, create, update, pause, delete campaigns
- **Ad Sets**: list, get, create, update, pause ad sets with targeting configuration
- **Ads**: list, get, create, update, pause ads
- **Creatives**: list, get, create ad creatives with object story specs
- **Targeting**: search targeting options (interests, behaviors, demographics), get targeting suggestions
- **Insights**: get insights at account/campaign/ad set/ad level with breakdowns and date ranges
- **Audiences**: list, create custom audiences, get audience size
- **Images**: upload images by URL, list ad account images
